自之前用 Windows 有了一段時間，回到 NixOS 還是要感受陌生又熟悉的感覺。寫寫指令，看 Nix 有甚麼更新之類的事……。
我是為了玩遊戲換成 Windows。在你開始碎碎念說：「我的 GNU/Linux 系統可以跑遊戲呀」之前，
我只是對於為了玩遊戲設定一大堆東西感到疲倦，而現在我沒有在玩遊戲，自然也沒有使用 Windows 的需求。
看到虛擬的競爭、虛擬的排行與虛擬的成就感只是令我不安罷了。而相較其他 Linux 系統，NixOS 算是最舒適的，
因為可以直接 git clone 自己的 dotfiles 就暢行無阻。

除了回坑這件事外，我最近開始看 10X 開發者的直播，像是 Tsoding 和 René Rebe。我對於 René Rebe 的 Vi 編輯器少了語法色彩感到疑惑，
不過他說這是為了生產力，我看未來也來試試看，不過現在的話我可是賴在 Neovim 的語法色彩功能上。講到寫程式，我快沒想法了，
最近一個專案是嵌入式系統，我真的需要提升自己的檔次並找些有趣的專案做。
